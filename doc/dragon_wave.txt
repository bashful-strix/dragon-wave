		  Dragon Wave color scheme    by bashful_strix

Dragon Wave, a color scheme based on the kanagawa dragon theme.

==============================================================================
Color scheme options					dragon_wave-options*

						*g:dragon_wave_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:dragon_wave_transp_bg = 0
<
